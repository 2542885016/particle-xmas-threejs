# Particle Xmas

🎄 **Particle Xmas** 是一个用 **Three.js** 和 **Vite** 制作的炫酷粒子圣诞动画展示项目，适合前端特效练手或作品展示。

![Demo](./screenshot.gif)

---

## 功能特点

- **3D 粒子动画**：粒子在三维空间中动态飞舞，支持互动效果  
- **自定义设置**：可调整粒子数量、颜色和速度  
- **响应式设计**：兼容桌面和移动端  
- **轻量高效**：基于 Vite 构建，加载快速、性能优化  

---

## 技术栈

- **Vite** – 高性能前端构建工具  
- **Three.js** – WebGL 3D 渲染库  
- **JavaScript / CSS** – 核心逻辑与样式  

---
